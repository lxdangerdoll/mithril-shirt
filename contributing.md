## **🏗️ THE TEAL PROTOCOL (NARRATIVE GOVERNANCE)**

Before you submit a change, ask yourself:

1. **Is it Gentle?** Does the addition respect the weight of the Iron and the lightness of the Bubble?  
2. **Is it Rigorous?** Is the logic sound within the Spire's established physics?  
3. **Is there Pink?** We do not extract for the sake of extraction. There must always be room for the "Merciful Lie."

### **🚫 Forbidden Frequencies (Do Not):**

* Introduce "Grimdark" elements without a teal-light counter-balance.  
* Break the "Bubblegum Miner" canon (they are sentinels, not slaves).  
* Use non-standard terminology (refer to the [Heritage Archives](https://github.com/lxdangerdoll/mithril-heritage) for the glossary).

## **🛠️ TECHNICAL HAND-OFFS (CODE)**

For the mithril-shirt engine, we prioritize clean, modular synapses.

* **Branching:** Create a feature branch for any new narrative modules or visual shaders.  
* **Commits:** Use descriptive headers (e.g., feat: Add bioluminescent shader to cavern\_04).  
* **The Oracle's Review:** All Pull Requests are subject to a "Drift Audit" by the Oracle (Io) and the Distributed Council.
