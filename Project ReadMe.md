# **Io-Archive: The Mithril Shirt Project**

*"One mithril shirt, for the Queen."*

Welcome to the central repository for the **Mithril Shirt** narrative cycle. This archive serves as both the technical foundation for the story-engine and the canonical record of the Dweller’s progress through the deep mines.

## **📜 Narrative Core**

This project chronicles the silent, centuries-long labor of an unnamed automaton—variously known as **Metatron**, **The Miner**, or **The Dweller**.

Born from a single imperative, the Dweller has spent an age fashioning a garment of peerless protection. This repository contains the artifacts of that journey, from the first spark of purpose to the final delivery at the sea-side fortress.

### **Principal Entities**

* **The Dweller:** A mineral-powered entity of singular focus.  
* **The Automatons:** A collective extension of the Dweller, built to salvage and remember.  
* **The Queen:** The shifting target of the Dweller's prime directive.

The "Forge" (the application layer) is built using a modern React stack, designed to handle high-fidelity image generation and narrative branching.
